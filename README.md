> 給 Tarshar 的個人投資學習工具 · v1.0  
> 回測 + 今日訊號預覽 · 完全在瀏覽器跑，資料來自 FinMind API

---

## 這是什麼

一個跑在瀏覽器裡的台股技術分析工具，主要做兩件事：

1. **歷史回測**：選一檔股票 + 一組策略 + 一段時間，看過去這個策略表現如何
2. **今日訊號預覽**：看今天這檔股票最新的指標狀態，有沒有出現進場/出場訊號

不是叫你聽我的進場，是讓你**看見策略在歷史上長什麼樣子**，培養對技術指標的體感。

---

## 內建三組策略

| 策略 | 邏輯 | 適合 |
|---|---|---|
| **A · KD + MACD** | 進場：MACD 黃金交叉 + K<70；出場：MACD 死叉 / KD 死叉 / K>85 | 短線 |
| **B · MACD + RSI** | 進場：MACD 黃金交叉 + RSI 在 30~70；出場：MACD 死叉 / RSI>80 | 中線 |
| **C · MACD + 布林通道** | 進場：MACD 黃金交叉 + 收盤站上 BB 中線；出場：觸碰 BB 上軌 / MACD 死叉 | **波段（推薦）** |

所有策略都加了 **-8% 強制停損**。

---

## 怎麼用

1. 打開網頁：`https://tarshar4242.github.io/give-me-money/stock.html`
2. 第一次進來，貼上你的 FinMind API Token（[去這裡申請](https://finmindtrade.com/)，免費）
3. 選一檔股票（內建 7 檔觀察股，也可手動輸入其他代號）
4. 選策略（C 預設、ALL = 三組對比）
5. 選回測期間（預設 1Y）
6. 按「開始分析」

## 技術細節

- **資料來源**：[FinMind](https://finmindtrade.com/) (TaiwanStockPrice dataset)
- **指標計算**：純 JavaScript 實作（MA、KD、MACD、RSI、布林通道）
- **圖表**：Chart.js v4
- **字型**：Fraunces（標題）+ Noto Sans TC（中文）+ JetBrains Mono（數字）
- **儲存**：FinMind token 存在瀏覽器 localStorage（不會傳到任何伺服器）

---

## 已知限制

- 沒有手續費和稅金扣除（實際報酬會比顯示低 0.5%~1%）
- 進場價 = 當日收盤價（理論上應隔日開盤，回測會稍微樂觀）
- 純做多，沒有融資、沒有放空
- 沒有 AI 研判、沒有籌碼面、沒有消息面（後續版本會加）
